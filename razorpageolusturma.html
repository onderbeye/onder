En Başta Yazılacak Komutlar

Yeni razor page uygulaması oluşturma :
dotnet new razor -o UygulamaAdi -f net5.0
cd UygulamaAdi
startup.cs içerisinde aşağıdaki satırı sil
app.UseHttpsRedirection();

properties/launchSettings.json içerisinde "UygulamaAdi" bloğuna aşağıdaki satırı ekle
"useSSL":false,

Projeyi derlemek için
dotnet build

Projeyi çalıştırmak için
dotnet run

// Hot reload aktive etmek için
dotnet add package Microsoft.AspNetCore.Mvc.Razor.RuntimeCompilation --version 5.0.15

services.AddRazorPages().AddRazorRuntimeCompilation(); //startup.cs içerisinde düzenle

CLI (Command Line Interface)

Sayfaya Database Bağlama

dotnet add package Microsoft.EntityFrameworkCore --version 5.0.15

dotnet add package Microsoft.EntityFrameworkCore.Design --version 5.0.15

dotnet add package Pomelo.EntityFrameworkCore.MySql --version 5.0.4

dotnet tool install --global dotnet-ef --version 5.0.15
dotnet ef dbcontext scaffold "Server=localhost;User=root;Database=Dosya Adı_db" "Pomelo.EntityFrameworkCore.MySql" -o Model -f
